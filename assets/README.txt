App Icon
========

Place your app.ico file here for the installer and EXE to use.

The icon should be a Windows .ico file with multiple sizes (16x16, 32x32, 48x48, 256x256).

To convert your PNG/SVG to .ico:
- Use an online converter: https://convertio.co/png-ico/
- Or use ImageMagick: magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 app.ico
